# NC News

To run this web app locally:

1. Clone this repo:

```bash
git clone https://github.com/RmPillar/NC-News-FE.git

cd NC-News-FE
```

2. Install dependencies:

```bash
npm install
```

3. Lanch the app:

```bash
npm start
```
